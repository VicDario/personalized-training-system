# Tareas pendientes según el enunciado

1. Implementar un menú por consola para operar el sistema.
2. Permitir crear ejercicios desde el programa.
3. Permitir actualizar ejercicios existentes.
4. Permitir eliminar ejercicios.
5. Permitir consultar la información completa de un ejercicio.
6. Permitir buscar ejercicios por nivel de intensidad.
7. Generar una rutina solicitando la cantidad de ejercicios.
8. Generar una rutina solicitando el nivel de intensidad deseado.
9. Calcular y mostrar el tiempo estimado total de la rutina generada.
10. Incorporar todos los datos obligatorios de cada ejercicio:
   - código identificador
   - nombre
   - tipo de ejercicio
   - nivel de intensidad
   - tiempo estimado en minutos
   - descripción
   - semana en la que fue usado por última vez
11. Evitar repetir ejercicios en rutinas de dos semanas consecutivas para un mismo cliente.
12. Asociar cada rutina a un cliente.
13. Aplicar correctamente abstracción, encapsulamiento, herencia y polimorfismo en el diseño.
14. Elaborar el diagrama de clases de la solución.
15. Preparar el informe con portada, introducción, descripción de la solución, diagrama de clases y conclusión.
