# Personalized Training System

Proyecto creado para ser presentado como Tarea 1 de la asignatura Paradigmas de Programación UNAB - Ingeniería en Computación e Informática.

# Instrucciones de ejecución

Este proyecto está escrito en C++ y puede compilarse desde la terminal usando `g++`.

## Requisitos

- Tener instalado `g++`
- Abrir una terminal en la carpeta del proyecto

## Ejecutar en Linux

1. Abrir una terminal dentro de la carpeta del proyecto.
2. Compilar todos los archivos `.cpp` con este comando:

```bash
g++ *.cpp -o app
```

3. Ejecutar el programa:

```bash
./app
```

## Ejecutar en Windows

1. Abrir `PowerShell` o `CMD` dentro de la carpeta del proyecto.
2. Compilar todos los archivos `.cpp` con este comando:

```powershell
g++ *.cpp -o app.exe
```

3. Ejecutar el programa:

```powershell
./app.exe
```

## Nota

Si `g++` no es reconocido como comando, hay que instalar un compilador de C++ como `MinGW-w64` en Windows o `g++` en Linux y asegurarse de que esté agregado al `PATH`.
